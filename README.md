# Flutter Tasks

## بيانات الطالب

الاسم: علي مرتضي عبداللاه

## وصف المشروع

هذا المشروع يحتوي على مجموعة من التاسكات المكتوبة بلغة Dart و Flutter. التاسكات تغطي أساسيات البرمجة، التعامل مع المدخلات، القوائم، المجموعات، الخرائط، الدوال، الـ recursion، مفاهيم البرمجة كائنية التوجه، وبناء واجهة Flutter بسيطة.

## محتوى التاسكات

### Task 1: نظام تقدير الدرجات

الملف: `task1.dart`

- توليد درجة عشوائية من 0 إلى 100.
- طباعة الدرجة التي تم توليدها.
- تحديد التقدير بناء على الدرجة:
  - `A` إذا كانت الدرجة 90 أو أكثر.
  - `B` إذا كانت الدرجة 80 أو أكثر.
  - `C` إذا كانت الدرجة 70 أو أكثر.
  - `D` إذا كانت الدرجة 60 أو أكثر.
  - `F` إذا كانت الدرجة أقل من 60.

### Task 2: نظام إدارة درجات الطلاب

الملف: `task2.dart`

- إدخال عدد الطلاب من المستخدم.
- إدخال اسم كل طالب.
- إدخال عدد المواد لكل طالب.
- إدخال درجات المواد مع التحقق أن الدرجة بين 0 و 100.
- عرض قائمة تفاعلية تحتوي على:
  - عرض نتائج كل الطلاب.
  - البحث عن طالب بالاسم.
  - الخروج من البرنامج.
- حساب متوسط درجات كل طالب.
- عرض التقدير النهائي للطالب حسب المتوسط.

### Task 3: التعامل مع Sets و Maps والدوال

الملف: `task3.dart`

- استخدام `Set` لتخزين أسماء الطلاب بدون تكرار.
- إضافة طلاب إلى المجموعة.
- طباعة أسماء الطلاب باستخدام recursion.
- استخدام `forEach` و lambda expressions.
- دمج أكثر من `Set` باستخدام spread operator.
- استخدام `Map<String, List<double>>` لتخزين درجات الطلاب.
- حساب متوسط درجات كل طالب.

### Task 4: البرمجة كائنية التوجه OOP

الملف: `task4.dart`

- إنشاء كلاس `Person` يحتوي على:
  - اسم الشخص.
  - متغير خاص للعمر.
  - getter و setter للعمر.
  - متغير static باسم الجامعة.
- إنشاء كلاس `Employee` يرث من `Person`.
- إنشاء abstract class باسم `Skills`.
- إنشاء كلاس `Developer` يرث من `Employee` ويطبق `Skills`.
- تطبيق مفاهيم:
  - Encapsulation.
  - Inheritance.
  - Abstraction.
  - Interfaces.
  - Method overriding.

### Task 5: تطبيق Flutter UI

المجلد: `task_5`

- إنشاء تطبيق Flutter باستخدام Material Design.
- استخدام `AppBar` يحتوي على عنوان وأيقونة إشعارات.
- عرض صورة شخصية دائرية.
- عرض نص ترحيبي.
- عرض خمس أيقونات نجوم.
- إنشاء حقلي إدخال للبريد الإلكتروني وكلمة المرور.
- إضافة زر لإظهار وإخفاء كلمة المرور.
- إضافة زر `Submit`.
- إضافة `FloatingActionButton`.
- عرض `SnackBar` يحتوي على البريد الإلكتروني عند الضغط على زر الإرسال.


```bash
cd task_5
flutter pub get
flutter run
```
