1. نظام تقدير الدرجات البسيط (task1.dart)
برنامج يعتمد على المنطق البرمجي الأساسي لتصنيف الدرجات.

المفاهيم المستخدمة: Random من مكتبة dart:math وتطبيق جمل if-else.

الوظيفة: توليد درجة عشوائية وتحديد التقدير (A, B, C, D, F) بناءً على قيمتها.

2. نظام إدارة الطلاب التفاعلي (task2.dart)
تطبيق CLI (Command Line Interface) متكامل لإدارة بيانات الطلاب ودرجاتهم.

المفاهيم المستخدمة: stdin لإدخال البيانات، القوائم Lists.

الميزات:

إدخال عدد الطلاب وعدد المواد لكل طالب.

حساب المتوسط الحسابي (Average) للدرجات.

قائمة (Menu) لعرض كل النتائج أو البحث عن طالب محدد بالاسم.

3. التجميعات والبرمجيات الوظيفية (task3.dart)
استعراض لقوة المجموعات والوظائف المتقدمة في Dart.

المفاهيم المستخدمة: Set (لمنع تكرار الأسماء)، Map (لربط الطلاب بالدرجات)، و Recursion.

الميزات: استخدام الـ Lambda expressions، دمج المجموعات عبر Spread operator (...).

4. البرمجة كائنية التوجه (task4.dart)
تطبيق عملي لمبادئ الـ OOP الأساسية.

المفاهيم المستخدمة:

Encapsulation: حماية البيانات باستخدام private variables والـ getters/setters.

Inheritance: استخدام الوراثة بين الكلاسات (Person -> Employee).

Abstraction & Interfaces: تنفيذ الكلاسات المجردة والواجهات البرمجية.

Static Members: استخدام المتغيرات الثابتة على مستوى الكلاس مثل universityName.
